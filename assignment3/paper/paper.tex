\begin{abstract}
A Toronto marriage licence is a legal document required for couples
wishing to get married within Ontario. It must be obtained from a
Service Ontario location or municipal office, and is valid for 90 days
from the date of issue. The application process requires both parties to
provide valid identification and may involve a fee. Once issued, the
licence permits a legally recognized wedding ceremony, but does not
serve as proof of marriage, which requires a separate certificate.
\end{abstract}

\subsection{Model set-up}\label{model-set-up}

Define \(y_i\) as the number of seconds that the plane remained aloft.
Then \(\beta_i\) is the wing width and \(\gamma_i\) is the wing length,
both measured in millimeters.

\begin{align} 
y_i|\mu_i, \sigma &\sim \mbox{Normal}(\mu_i, \sigma) \\
\mu_i &= \alpha + \beta_i + \gamma_i\\
\alpha &\sim \mbox{Normal}(0, 2.5) \\
\beta &\sim \mbox{Normal}(0, 2.5) \\
\gamma &\sim \mbox{Normal}(0, 2.5) \\
\sigma &\sim \mbox{Exponential}(1)
\end{align}

We run the model in R (R Core Team 2023) using the \texttt{rstanarm}
package of Goodrich et al. (2022). We use the default priors from
\texttt{rstanarm}.
